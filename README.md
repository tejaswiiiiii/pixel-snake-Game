<h2 align="center">🎮 About Pixel Snake</h2>

<p align="center">
  <i>"Eat. Grow. Survive."</i>
</p>

<p>
<b>Pixel Snake</b> is a retro-inspired classic snake game built with 
<b>Node.js</b>, <b>Express.js</b>, <b>HTML</b>, <b>CSS</b>, and <b>JavaScript</b>.
The game delivers smooth controls, score tracking, responsive gameplay, and an arcade-style pixel experience.
</p>

---

<h2>✨ Features</h2>

<ul>
  <li>🐍 Smooth Snake Movement</li>
  <li>🍎 Food Collection System</li>
  <li>🏆 Live Score Tracking</li>
  <li>💀 Game Over Detection</li>
  <li>📱 Responsive Design</li>
  <li>⚡ Fast Backend with Node.js + Express</li>
</ul>
